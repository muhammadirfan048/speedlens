# SpeedLens

A minimal, dark-first internet speed test built with Next.js App Router, React, TypeScript, TailwindCSS, Framer Motion, and Lucide Icons.

## Run

```bash
npm install
npm run dev
```

The test starts automatically on page load. The speed engine lives in `services/SpeedTestService.ts` and can be replaced later with LibreSpeed or custom servers without changing the UI.
